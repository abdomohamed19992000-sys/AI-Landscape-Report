# 🤖 AI Tools Landscape Analysis

## 📌 Overview
This project analyzes the AI tools ecosystem to uncover insights about market trends, tool distribution, pricing models, and popularity.

The goal is to provide a clear understanding of how AI tools are evolving and how they are distributed across different domains and categories.

---

## 🎯 Objective
- Analyze the growth of AI tools over time  
- Understand pricing strategies (Free vs Paid)  
- Identify the most popular tools and developers  
- Explore tool distribution by domain and intelligence type  

---

## 🧹 Data Preparation
- Cleaned and structured dataset using Excel / Power Query  
- Handled missing values and inconsistencies  
- Prepared data for analysis and visualization  

---

## 🗄️ Data Analysis
- Analyzed tool growth by year  
- Compared pricing distribution (Free vs Paid)  
- Evaluated popularity metrics  
- Identified top tools and developers  

---

## 📊 Data Visualization (Power BI)
Developed an interactive dashboard including:
- Total number of AI tools  
- Average popularity votes  
- Growth trends over years  
- Pricing distribution  
- Tools by domain and intelligence type  
- Top 10 tools by popularity  

---

## ⚙️ Tools & Technologies
- Power BI (Dashboard & Data Modeling)  
- Excel / Power Query (Data Cleaning)  
- SQL (Data extraction & analysis)  
- Python (Optional: Data processing & analysis)  

---

## 📈 Key Insights
- Rapid growth in AI tools over recent years  
- Significant portion of tools follow freemium pricing models  
- Clear dominance of specific domains (e.g., NLP, Automation)  
- Popularity is strongly linked to accessibility and pricing
