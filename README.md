# 📊 Sales Data Analysis

## 📌 Overview
This project focuses on analyzing sales data to extract meaningful insights and support business decision-making. The workflow includes data cleaning, analysis, and data visualization.

---

## 🎯 Objective
To transform raw and unstructured sales data into clear insights that help understand business performance.

---

## 🧹 Data Cleaning
- Removed duplicates
- Handled missing values
- Standardized date formats
- Ensured data consistency

---

## 🗄️ Data Analysis (SQL)
- Calculated total sales
- Identified top-selling products
- Analyzed monthly sales trends
- Aggregated data for reporting

---

## 📊 Data Visualization (Power BI)
Built an interactive dashboard that includes:
- Total Sales
- Profit
- Sales by Month
- Top Products

---

## ⚙️ Tools & Technologies
- Excel (Data Cleaning & Preparation)
- Python (Pandas for data processing)
- SQL (Data querying & analysis)
- Power BI (Dashboard & visualization)

---

## 📈 Key Insights
- Identified top-performing products contributing to revenue
- Discovered seasonal trends in sales
- Improved visibility of key business KPIs
